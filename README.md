# 100 Готовых решений программ на языке Си
В данном репозитории представлены решения примеров программ на языке Си из книжки Кольцова Д. М.
Эти примеры можно использовать для ознакомления и изучения Си.

###Для компиляции программ использовался GNU Compiler Collection(gcc)
    gcc 1.c -o 1
###Для компиляции с библиотекой math.h используется приставка -lm
    gcc 1.c -o 1 -lm
